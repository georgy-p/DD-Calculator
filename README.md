# Drag'n'Drop calculator
DD-Calcultor is an SPA application with drag'n'drop function. Choose any order of elements to build your own calculator.

## How to install
1. Clone the repository.
2. In the root directory: `$ make install`.
3. Run it on you localhost: `$ make start`.
